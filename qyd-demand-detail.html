清华大学环境学院：全自动高精度水质监测系统采购需求

发布时间：2026-04-01 · 浏览次数：1,248 次

项目背景

为支持“十四五”国家重点实验室绿色升级计划，清华大学环境学院拟采购一套全自动水质监测系统，用于长期监控校内循环水系统的节能指标与排放标准。本项目受国家环保部专项资金支持。

核心需求

1. **监测精度**：需支持 COD、氨氮、总磷等 8 项参数实时监测，误差率低于 1%。

2. **绿色能源**：设备需支持太阳能辅助供电，具备超低功耗待机模式。

3. **系统联动**：需提供标准 API 接口，能够无缝接入校级“绿色校园管理后台”。

4. **售后要求**：提供不少于 3 年的现场技术支持及系统终身在线升级。
